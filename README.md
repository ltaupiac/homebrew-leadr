## Leadr Homebrew Tap

This repository contains the necessary formula to install [leadr](https://github.com/ll-nick/leadr) via Homebrew.

### Installation

To install the leadr formula from this tap, run the following command:

```bash
brew tap ll-nick/leadr
brew install leadr
```
